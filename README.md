# Gym Karo — FlipFit

Gym Karo (FlipFit) is a modern fitness web application that helps users find classes, book sessions with trainers, track progress, and manage memberships. It is built to be mobile-friendly, fast, and easy to extend — ideal for gyms, trainers, and end users who want a single place to manage fitness activities.

## Project Overview
Gym Karo provides:
- An easy onboarding flow for users and trainers
- Class scheduling and booking
- Workout plans and progress tracking
- Membership and payment management
- Admin dashboard to manage users, trainers, classes, and analytics
